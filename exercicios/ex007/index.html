Exemplo de outras tags

Mark Del e Ins

A tag mark é usada como marca texto

A tag del é para textos excluidos que podem ser lidos mas não devem ser considerados essa parte não vale de nada

A tag ins é o oposto do del e é para textos que devem ser lidos com atenção pois são importantes.

Big e Small

A tag big é para textos grandes contudo não é mais usada pois é forma e não tem semantica texto grande

Diferente da tag Big a tag small ainda é usada pois ainda existe sentido em letras pequenas e não só forma texto pequeno

Sub e Sup

A tag sup e sub são usadas para textos sobrescritos exemplo1 e subscritos exemplo2

Q blockquote e cite

A tag q é para citações onde o texto ja vem com as aspas e não apresenta quebra de linha Lorem ipsum dolor sit amet, consectetur adipisicing elit. Eius ullam optio ratione cum aut nihil. Fugit, consequuntur! Incidunt, non. Harum?

Existe outra tag para citações, a blockquote que é usada para citações em blocos e que usa uma quebra de linha por padrão

Lorem ipsum dolor sit amet consectetur adipisicing elit. Pariatur voluptas, aspernatur velit aut nobis fuga sequi totam dolorem molestiae? Consequuntur voluptates fugit, in inventore totam veritatis iure rerum quam id?

Existe ainda o cite que vai dentro do blockquote e te permite colocar o link para o texto original
